# Dice Roll Simulator & Adventure Game

A text-based dice rolling and dungeon adventure game.

## Features
- Choose different dice: D6, D8, D12, D20, or a custom dice.
- Roll multiple dice at once.
- Multiplayer support: each player can take turns.
- Player stats: health, damage, defense.
- Inventory system: swords, shields, health potions.
- Random encounters: fight enemies or find treasure.
- Combat system with turn-based battles.

## Usage
Run the script:
```bash
python Dice-Roll\ Simulator.py
