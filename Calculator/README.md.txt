# Calculator

A basic Python calculator with a menu-driven interface.

## Features
- Addition
- Subtraction (choose n1 - n2 or n2 - n1)
- Multiplication
- Exponentiation
- Division (n1/n2 or n2/n1)
- Floor Division
- Modulo

## Usage
Run the script:
```bash
python Calculator.py
